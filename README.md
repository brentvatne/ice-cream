# Vancouver Ice Cream Festival app example written in Expo UI

This is a demo app built with Expo UI, showcasing native components and Expo Router native tabs. It uses data for the Vancouver Ice Cream Festival sourced from [Noms Magazine](https://nomsmagazine.com/vancouver-ice-cream-festival/).

The creators of this app are not affiliated with the festival or Noms Magazine.

## Get started

1. Install dependencies

   ```bash
   bun install
   ```

2. Build and start the app

   ```bash
   npx expo run:ios
   ```
